Chậu cây thông minh - Project KHKT 2025
Google Docs documentation [here](https://docs.google.com/document/d/1plYU-8LHdzI_1yI13td3qCeDksVRaw9wgZwrKLIwX0k/edit?usp=sharing)

# Installation
Download và mở trong Arduino IDE, đảm bảo các file thư viện (cpp, c, h) chung thư mục với file .ino

# Thư viện ngoài
Ngoài các thư viện vốn đi kèm với file .zip, cần cài thêm vài thư viện bổ trợ kh thông qua Arduino IDE.
Mở Library Manager của Arduino IDE, cài thêm các thư viện sau:
- ArduinoHttpClient (by Arduino)
- Adafruit Unified Sensor (by Adafruit)
- ArduinoJson (by Benoit Blanchon...)
- DHT kxn (by Adafruit)
- DHT sensor library (by Adafruit)
- WebSockets (by Markus Sattler)
- xiaozhi-mcp (by toddpan)
